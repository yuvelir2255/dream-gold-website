# План: Премиум-апгрейд P0 + P1 (дизайн до уровня люкс-брендов)

Дата: 2026-06-16. Основа — анализ референсов (Cartier, Tiffany, VCA, Bulgari, Messika,
Pandora, Swarovski, Mejuri). Цель: поднять ощущение премиальности. НЕ копируем чужое — берём принципы.

Скиллы: `ui-ux-pro-max` + `frontend-design` (обяз.), плюс `web-design-guidelines`,
`react-best-practices`, `react-view-transitions`, `composition-patterns`.

Ограничения (не нарушать): нет публичных цен; вариант А (без чужих брендов); шрифт EB Garamond
(не Cormorant — нет кириллицы); без 3D; reduced-motion безопасно (не менять структуру дерева
по флагу — только длительность/диапазоны); не ломать гидрацию.

## P0 — максимальный эффект
1. **Ритм и типографика** — больше «воздуха», крупнее заголовки, единая шкала отступов.
   Файлы: `Section.tsx`, `SectionHeading.tsx`.
2. **Премиум-карточка товара** — hover-свап на 2-й кадр, больше воздуха, сдержанный hover.
   Файлы: `ProductCard.tsx` (и `FeaturedWorks` использует её же).
3. **Фото-готовый layout** — подготовить hero/витрину под реальные фото (после получения фото).
   Файлы: `Hero.tsx`, `FeaturedWorks.tsx`. *(делаем, когда будут обработанные фото)*

## P1 — сильно усиливают
4. **Слой доверия/мастерства на главной:** секция «Savoir-faire / мастерская» (процесс) +
   полоса сервис-обещаний (доставка/возврат/сертификат/упаковка/консультация).
   Файлы: новые `Craft.tsx`, `ServicePromise.tsx`; вставка в `src/app/[locale]/page.tsx`.
5. **Микроанимации и переходы:** плавные page transitions (`react-view-transitions`),
   отшлифованный hover, sticky-CTA на карточке изделия.
6. **«Избранное» (wishlist)** — без бэкенда, хранение в браузере (localStorage):
   контекст-стор + кнопка-сердце на карточках + страница `/favorites`.
   Файлы: новый `src/components/favorites/*`, иконка на `ProductCard`, страница favorites, ссылка в шапке.
7. **Страница «Как проходит заказ»** — шкала этапов (заявка → расчёт → предоплата → производство →
   доставка). Файлы: новая `src/app/[locale]/process/page.tsx` (+ тексты, ссылки из `/order` и нав).

## Порядок батчей (по этапам — для качества)
- **Batch 1 ✅:** ритм/типографика (Section/SectionHeading) + премиум-карточка (hover-свап).
- **(доп.) ✅:** фото-ориентированный hero на весь экран + editorial-секция «Лукбук» + система `PhotoSlot`.
- **Batch 2 ✅:** полоса сервис-обещаний (`ServicePromise`) + секция «Savoir-faire» (`Craft`).
- **Batch 3 ✅:** «Избранное» (wishlist в localStorage: сердце на карточках + счётчик в шапке + `/favorites`).
- **Batch 4 ✅:** страница «Как проходит заказ» `/process` (timeline 6 этапов) + ссылка с главной.
- **Batch 5 ✅:** плавные переходы между страницами (Framer Motion, opacity-fade — стабильнее экспериментального
  React View Transitions: не конфликтует с Lenis/Suspense, типизирован; нативный VT можно включить позже) +
  sticky-CTA на карточке (десктоп — sticky-колонка деталей, мобайл — плавающая кнопка) + hover-зум главного фото галереи.
- **Batch 6 (осталось):** интеграция реальных фото (замена `PhotoSlot` → `<Image>`), когда будут готовы.
