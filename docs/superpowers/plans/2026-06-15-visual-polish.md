# Визуальная полировка — премиум-движение и эффекты

> Доп-проход к Этапу 1–2. Цель: «дорогое», современное, кинематографичное ощущение.
> Дизайн-направление (ui-ux-pro-max): Liquid Glass + наша палитра (чёрный/золото/слоновая кость),
> плавные анимации 400–600 мс, мягкий блюр. Применяем ТОЧЕЧНО, со вкусом.

## Принципы
- Уважать `prefers-reduced-motion` — при включённой опции отключать параллакс/3D/Lenis.
- Глассморфизм только там, где не страдает читаемость (достаточная непрозрачность).
- Анимации на `transform`/`opacity` (не дёргать layout). Тяжёлый 3D — лениво и не на мобиле.
- Никаких эмодзи как иконок; контраст текста ≥ 4.5:1; `cursor-pointer` на кликабельном.

## Библиотеки (чистый JS, без нативных бинарников)
- `lenis` — плавный скролл.
- `three`, `@react-three/fiber`, `@react-three/drei` (+ `@types/three`) — 3D-камень в hero.
- Framer Motion — уже установлен (параллакс/скролл/появления).

## Батч P1 — основа движения + глассморфизм
- `SmoothScroll` провайдер (Lenis) в `[locale]/layout`, с отключением при reduced-motion.
- `Header` → «стеклянная» прилипающая шапка (`sticky top-0`, `backdrop-blur`, тёмный полупрозрачный фон, тонкая золотая граница при скролле). Сделать клиентским с реакцией на скролл.
- Кнопки (`Button`) — золотой подсвет/линия при наведении (без сдвига layout).
- `ProductCard` / featured — мягкий зум фото при наведении (`scale`, overflow-hidden).
- Улучшить `Reveal` — варианты (fade-up, stagger) для секций.

## Батч P2 — параллакс + сторителлинг + стекло-фильтры
- `Hero` — параллакс: фон/градиент и заголовок двигаются с разной скоростью (Framer Motion `useScroll`+`useTransform`). Лёгкий, не «уезжающий».
- `CreateYourOwn` — scroll-сторителлинг: вертикальная золотая линия «прорисовывается» по мере скролла (`scaleY` по прогрессу), шаги появляются по очереди.
- `CatalogFilters` — стеклянная панель (`backdrop-blur`, светлый полупрозрачный фон, граница `border-line`).

## Батч P3 — 3D-камень в hero
- `components/three/Gem.tsx` (`'use client'`): сцена r3f — гранёный камень (октаэдр/икосаэдр) с премиальным материалом (`MeshTransmissionMaterial`/преломление) + мягкий `Environment`, медленное вращение.
- Подключать через `dynamic(() => import(...), {ssr:false})`; рендерить только на десктопе и при отсутствии reduced-motion; запасной вариант — без 3D (чистый hero). Низкополигональный, пауза вне вьюпорта.
- Камень — деликатный акцент (не перекрывает текст, читаемость заголовка сохраняется).

## Проверка
- `npm run build` зелёный после каждого батча.
- Скриншоты главной (десктоп + мобайл): hero, шаги, шапка при скролле.
- Проверить reduced-motion (анимации выключаются) и что текст читаем.

## Не входит
- Реальные фото/видео (добавит владелец) — эффекты заиграют сильнее с настоящими медиа.
