## 1. О бренде (контекст)

**Dream Gold** — ювелирный дом полного цикла (Украина).

- Своё производство (отец, Украина) — сильная сторона = качество и контроль.
- Фокус: **индивидуальные заказы** + изделия **в стиле известных домов**
  (внутри ориентир — Cartier, Tiffany, Van Cleef, Graff; на сайте бренды НЕ называем).
- Что умеем: изделие по любому фото или идее; переплавка старого золота клиента
  в новое изделие; пробы 585 и 750; природные и лабораторные бриллианты с сертификатами.
- География: доставка по Украине и по всей Европе.
- Каналы сейчас: Telegram-канал + Instagram (ведёт мама) + Telegram Mini App (каталог).
- Лето 2026: офлайн-студия/шоурум в Харькове (помещение куплено, идёт ремонт).
- Через ~месяц: профессиональная съёмка производства и процесса создания изделий
  (контент для сайта — добавим позже).
- Приоритет — **сайт** (важнее Mini App), в едином стиле, но максимально стильный и новый.

## 2. Цели сайта

1. Максимально профессиональное, «дорогое» впечатление (не обычный магазин).
2. SEO — чтобы бренд находили в Google (закладываем с первого дня).
3. Принимать индивидуальные заявки онлайн + вести людей в будущий шоурум.
4. «Не как у всех» — современный, модный, кинематографичный вид.

## 3. Зафиксированные решения

| Тема | Решение |
|---|---|
| Платформа | Свой сайт на Next.js (не Shopify) — ради уникальности и SEO |
| Концепция | **B + A**: каркас «заказ/процесс в центре», эстетика «кино-премиум maison» |
| Языки | **2 языка: украинский (основной) + английский.** Архитектура i18n — с запасом, чтобы легко добавить русский и др. позже |
| Цены | Публично НЕ показываем (расчёт индивидуальный) |
| Бренды | Не называем напрямую — формулировка «в стиле известных ювелирных домов» (вариант А) |
| Заявки | На старте приходят **только в Telegram**. Почта/CRM — Этап 2 |

| Фото | Только **свои изделия** + грамотная подача + ИИ-обработка **своих** фото. Чужие фото брендов НЕ используем |
| Палитра/шрифты | Оставляем существующую дизайн-систему (см. §6) |

## 4. Концепция B + A (главная идея)

Сайт ощущается как **глянцевый журнал ювелирного дома** (эстетика A: крупная
типографика, много «воздуха», атмосферные фото/видео, медленные плавные анимации),
но **построен вокруг индивидуального заказа** (каркас B: путь «создай своё» вынесен
вперёд, акцент на процессе и своём производстве).

Почему так: это прямо отражает бизнес (бенспоук), даёт больше заявок и **меньше
зависит от количества фото** — премиум создаётся подачей, а не «стеной» снимков.
Точечно добавим современные акценты (элементы концепции C), чтобы было свежо.

**Арт-дирекшн (по референсам владельца):** мягкий природный свет, малая глубина
резкости, изделие-герой; тёплые природные фактуры (камень, песок, крафт, шёлк);
макро-детали + кадры «на модели». Тёплая земляная гамма, совпадает с палитрой бренда.

## 5. Структура сайта (Этап 1)

**Страницы:**

1. **Главная** — см. §5.1.
2. **Каталог / Коллекции** — галерея изделий (CMS), фильтры: тип, металл, проба,
   «в стиле…». Фото + видео.
3. **Карточка изделия** — крупные медиа, описание, кнопка «Хочу такое же» → форма заказа.
4. **Индивидуальный заказ** — пошаговая форма: идея/фото-референс → металл → проба →
   размер → контакты. Заявка → в Telegram.
5. **О бренде / Мастерская** — история, процесс, своё производство (место под будущие
   профи-фото производства).
6. **Контакты** — Telegram, телефон, Instagram, форма. (Шоурум добавим после открытия.)
7. **Определитель размера кольца** — интерактивная фишка «не как у всех».

Отдельной вкладки «Отзывы» НЕТ — отзывы блоком на главной.

### 5.1. Главная — порядок блоков (версия B+A)

1. **Шапка** — логотип, меню, переключатель UA / EN.
2. **Кино-обложка (Hero)** — атмосферное фото/видео на весь экран, слоган про создание
   своего изделия, 2 кнопки: «Создать изделие» / «Смотреть работы».
3. **Путь заказа «Создай своё»** — интерактивные шаги (идея/фото → металл/камень →
   расчёт → изготовление). Вынесено высоко — это сердце концепции B.
4. **Избранные работы** — 3–6 лучших изделий, поданы как произведения (эстетика A).
5. **Что мы можем** — сильные стороны: по фото/идее, в стиле известных домов,
   переплавка вашего золота, природные и лабораторные бриллианты, пробы 585/750,
   доставка Украина + Европа.
6. **Почему нам доверяют / Своё производство** — акцент на собственном производстве
   (место под будущие фото процесса).
7. **Отзывы** — секцией.
8. **Финальный призыв** — крупный блок «оставить заявку».
9. **Подвал** — контакты, соцсети, языки.

## 7. Стратегия фотографий

**Подход (согласован):** свои изделия + грамотная подача + ИИ только по своим фото.

- Мама снимает реальные изделия на чистом белом фоне (2–3 ракурса, дневной свет).
- ИИ по этим **своим** фото: вырезает предмет, ставит на модель, добавляет фон/сцену
  под стиль сайта, улучшает свет, апскейл. (Гайд по инструментам и промтам —
  отдельным документом, когда сайт будет готов.)
- Атмосферные/фоновые кадры (текстуры, руки, мастерская) — из легальных стоков,
  как «настроение», НЕ как «наши изделия».
- Чужие фото брендов НЕ используем (риск копирайта; и не нужно — свой путь даёт тот же вид).
- Каталог на CMS → заменить фото на профи-версии после съёмки в Харькове = быстро,
  без переделки сайта.

**Старт:** достаточно 3–4 изделий, снятых на белом фоне, чтобы обкатать процесс.

## 8. Технический стек

- **Next.js (App Router) + TypeScript + Tailwind CSS**
- **shadcn/ui** — компоненты (`npx shadcn@latest init`)
- **Framer Motion** — плавные «дорогие» анимации
- **next-intl** — мультиязычность (UA по умолчанию + EN; задел под другие языки)
- **Sanity** — headless CMS для каталога
- **Деплой:** Vercel + собственный домен (будет куплен)
- **GitHub:** публичный репозиторий (портфолио). Секреты только через `.env` + `.gitignore`

## 9. Объём Этапа 1 (что входит / что НЕ входит)

**Входит:** многоязычный (UA+EN) сайт-витрина · каталог через CMS · карточка изделия ·
форма индивидуального заказа (→ Telegram) · страницы «О бренде» и «Контакты» ·
определитель размера кольца · SEO-базис · деплой на Vercel.

**НЕ входит (позже):**
- **Этап 2:** база данных · CRM-панель (заказы, клиенты, статусы) · онлайн-предоплата
  (укр. эквайринг: WayForPay / LiqPay / Fondy / Monobank) · дубль заявок на почту ·
  уведомления.
- **Этап 3:** личные кабинеты клиентов · продвинутая CRM · аналитика/дашборды ·
  блог/журнал для SEO · email-рассылки · программа лояльности.

## 10. Пошаговый план Этапа 1 (порядок работ)

1. **Каркас проекта.** Next.js + TypeScript + Tailwind, базовая структура папок,
   подключение дизайн-системы (цвета, шрифты как токены). Инициализация Git.
2. **i18n.** Подключаем next-intl, структура адресов (`/uk`, `/en`), переключатель языков.
3. **Общие части.** Шапка, подвал, базовые UI-компоненты (shadcn/ui), сетка/типографика.
4. **Главная.** Собираем блоки по §5.1 (сначала на «заглушках», потом наполняем).
5. **CMS Sanity.** Схемы каталога (изделие: фото, видео, тип, металл, проба, «в стиле»,
   описание на 2 языках). Подключение к сайту.
6. **Каталог + карточка изделия.** Список с фильтрами + страница изделия.
7. **Форма индивидуального заказа.** Пошаговая форма + загрузка фото-референса +
   отправка заявки в Telegram (через бота; токен — в `.env`).
8. **Страницы «О бренде» и «Контакты».**
9. **Определитель размера кольца.**
10. **SEO-базис.** Мета-теги, sitemap, человекочитаемые адреса, Open Graph, alt у фото.
11. **Контент.** Тексты (UA+EN), первые изделия в CMS, обработанные фото.
12. **Деплой.** Vercel + домен, проверка на телефоне и десктопе.

> Каждый шаг — отдельная небольшая задача; делаем по очереди, показываю результат,
> ты подтверждаешь — двигаемся дальше.
