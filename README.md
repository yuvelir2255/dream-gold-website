# Dream Gold — сайт ювелирного дома

Профессиональный сайт ювелирного дома **Dream Gold** (Украина): индивидуальные
изделия на заказ, изделия в стиле известных домов, изготовление по фото/идее.
Своё производство. Доставка по Украине и Европе.

## О проекте

- **Концепция:** «кино-премиум maison», построенный вокруг индивидуального заказа (B+A).
- **Языки:** украинский (основной) + английский (с заделом на другие).
- **Цены:** не показываем публично — индивидуальный расчёт под заказ.

## Стек

- Next.js (App Router) + TypeScript + Tailwind CSS
- shadcn/ui · Framer Motion
- Sanity (CMS для каталога)
- next-intl (мультиязычность)
- Деплой: Vercel

## Этапы

1. **Этап 1 (текущий):** сайт-витрина + каталог (CMS) + форма заказа (→ Telegram) + SEO.
2. **Этап 2:** база данных + CRM + онлайн-предоплата (укр. эквайринг).
3. **Этап 3:** личные кабинеты, аналитика, блог/журнал, рассылки, лояльность.
