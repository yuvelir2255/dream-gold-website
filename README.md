# Dream Gold — сайт ювелирного дома

Профессиональный многоязычный сайт ювелирного дома **Dream Gold** (Украина): индивидуальные
изделия на заказ, изготовление по фото/идее, изделия в стиле известных домов, переплавка золота.
Своё производство. Доставка по Украине и Европе.

Эстетика — «кино-премиум maison»: крупная serif-типографика, много «воздуха», атмосферные
медиа, плавные анимации. Сайт построен вокруг индивидуального заказа.

## Возможности (Этап 1)

- **Многоязычность** (укр/англ) на next-intl, с заделом на другие языки.
- **Фото-ориентированная главная:** hero на весь экран, editorial-лукбук, путь заказа,
  витрина работ, мастерская (savoir-faire), полоса сервис-обещаний, отзывы, финальный CTA.
- **Каталог и карточка изделия** через CMS **Sanity** (встроенная студия на `/studio`).
- **Форма индивидуального заказа** `/order` → заявка приходит в **Telegram**.
- **«Избранное»** (wishlist) — без бэкенда, хранение в браузере.
- **Страница «Как проходит заказ»** `/process` — этапы от заявки до доставки.
- **Система фото-слотов** (`PhotoSlot`) — места под будущие фото с подписями формата.
- Уважение `prefers-reduced-motion`, гидрация-безопасные анимации.

## Стек

- Next.js (App Router) + TypeScript + Tailwind CSS v3
- Framer Motion · Lenis (плавный скролл)
- Sanity (headless CMS для каталога)
- next-intl (мультиязычность)
- Деплой: Vercel (план)

## Запуск

```bash
npm install
npm run dev        # http://localhost:3000  (→ /uk)
```

Студия CMS: `http://localhost:3000/studio`

## Переменные окружения

Создайте `.env.local` (в Git не попадает):

```
NEXT_PUBLIC_SANITY_PROJECT_ID=...
NEXT_PUBLIC_SANITY_DATASET=production
NEXT_PUBLIC_SANITY_API_VERSION=2024-10-01

# Для заявок с формы заказа (необязательно — без них форма покажет ошибку отправки):
TELEGRAM_BOT_TOKEN=...
TELEGRAM_CHAT_ID=...
```

## Структура

- `src/app/[locale]/` — страницы: главная, каталог, карточка, `order`, `favorites`, `process`
- `src/app/api/order/` — серверный роут отправки заявки в Telegram
- `src/components/` — UI-компоненты (`ui`, `home`, `catalog`, `favorites`, `layout`)
- `src/sanity/`, `sanity.config.ts` — CMS
- `messages/` — переводы (`uk`, `en`)
- `docs/` — спеки, планы, гайд по ИИ-обработке фото
